# ⚔️ Marcos Adventure — Mobile

> *"Now you can defeat the professor on the bus, in class, or literally anywhere."*

---

## About

**Marcos Adventure Mobile** is the mobile version of the classic (and very serious) action game **Marcos Adventure** — built entirely with HTML, with an interface fully adapted for smaller screens and touch controls.

Same mission: take down **3 epic bosses** based on the professor's stickers, now in the palm of your hand.

> ⚠️ **Disclaimer:** Playing during class is entirely your responsibility. The developer is not liable for low grades or suspicious looks from the professor.

---

## What's Different in the Mobile Version

- **Touch controls** — fully adapted for touchscreen devices
- **Responsive layout** — optimized interface for smartphones and tablets
- **Same 3 bosses** from the desktop version, now at your fingertips
- **No installation** — opens directly in the mobile browser

---

## Getting Started

No app store. No APK. Just HTML.

```bash
# Clone the repository
git clone https://github.com/ChristopherDond/MarcosAdventureMobile.git
cd MarcosAdventureMobile

# Open index.html in your mobile browser
# Tip: host it on GitHub Pages and access it anywhere via link
```

## Tech Stack

- **HTML5** — game structure

No external libraries. Zero dependencies.
